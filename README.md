## Запуск проекта:
* Клонируйте проект на свой компьютер с Github с помощью команды:
```
git clone [https://github.com/amka-dev/React-Bikes.git]
```

* Установите зависимости с помощью команды:
```
npm install
```
* Запустите проект с помощью команды:
```
npm run start
```
